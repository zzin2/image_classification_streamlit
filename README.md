# 이미지 분류 앱
분류하고 싶은 이미지를 한 장/여러 장 업로드하거나, 직접 촬영한 이미지를 분류할 수 있습니다.

## 설치
``bash
pip install -r requirementsl.txt

# 실행
streamlit run app.py

# 기능
- 이미지 업로드
- Top-5 예측 출력
- 그래프로 시각화
